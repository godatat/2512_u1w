## nondegu-ruguru

2025年12月に開催された[Unity1Week](https://unityroom.com/unity1weeks/68)に投稿した作品です。

ゲームは[こちら](https://unityroom.com/games/nondegu-ruguru)から

当時のお題は「もうひとつ」となっており、自分は新人ホストになったずんだもんと一緒に楽しいコールをして「もういっぱい」をお客さんから引き出すタイピングゲームを制作しました。

<img src="./Docs/thumnail.png" width="50%">

> [!IMPORTANT]
> 平日仕事をしながら一週間でゲームを作る時間を捻出するのはとても大変で、実質12/27, 28の土日で製作しました。
>
> unity1weekでは綺麗に作るというよりは「雑なところがあっても何かひとつでも楽しい部分がある」というところを作り出したいと考えて取り組みました。
