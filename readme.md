# 言い訳。別の名をエクスキューズ

Unity1Weekは一週間で考えたゲームを形にするプロトタイピング的な表現の意味合いが強いと考えております。

私も綺麗に作るというよりは「雑なところがあっても何かひとつでも楽しい部分がある」というところを作り出したいと考えて取り組みました。

平日仕事をしながら一週間でゲームを作るのは時間を捻出するのもとても大変で、実質 12/27, 28の土日で製作しました。
